# Roblox username Checker
- Generate and test usernames on Roblox with a specific number of characters, and save them in a list.
- Choose and test a specific list of usernames.

<img style="width:50%" src="https://i.imgur.com/YaCAGyO.png"/>

- [!] This code will not run in python2.
- [+] You can use the code in Replit.

# Installing required packages:
```
pip install requests os colorama
```

# Running the script:
```
python3 main.py -- Linux
python main.py -- Windows
```
